# 46 Anarchy Launcher

A community-made Minecraft launcher for Paladium.

## Installation

```bash
npm install
npm start
```

## Build

```bash
npm run build    # Windows
npm run build:linux  # Linux
```

## Backend

The game server backend is available here:
https://github.com/Satanas1275/46anarchy-launcher-backend
